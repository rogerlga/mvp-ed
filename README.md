# MVP Engenharia de Dados

> Projeto com aplicação técnicas de **Engenharia de Dados** referente à entrega do MVP do curso de pós-graduação em *Ciência de Dados & Analytics* da PUC-Rio (2025).

Este trabalho apresenta uma análise exploratória de incidentes envolvendo sistemas de inteligência artificial, buscando compreender como esses eventos têm se manifestado e quais riscos podem representar em um cenário de uso cada vez mais amplo da tecnologia. A partir de uma base pública de incidentes e utilizando tecnologias de nuvem para organização, processamento e exploração dos dados, o estudo examina padrões gerais, identifica tipos relevantes de ocorrências e avalia aspectos como gravidade, vieses sociais e evolução temporal dos registros. Entre as questões que orientam a análise estão, por exemplo: se existem incidentes envolvendo morte de pessoas, se há evidências de que modelos de IA reproduzem vieses sociais como racismo ou homofobia, qual é a representatividade desse tipo de evento dentro do total analisado e se a taxa de incidentes tem aumentado ao longo dos anos. O objetivo é oferecer uma visão estruturada do tema, mostrando como determinados comportamentos da IA podem gerar impactos significativos e por que o monitoramento sistemático desses casos se tornou essencial.

Além de apresentar os resultados obtidos, o trabalho também discute suas limitações e oportunidades de aprimoramento, como a ampliação das fontes de dados analisadas, a aplicação de técnicas estatísticas mais robustas e o uso de modelos de linguagem para melhorar a classificação dos incidentes a partir de suas descrições textuais. Essa reflexão metodológica serve como base para orientar estudos futuros e tornar o monitoramento de incidentes mais preciso e abrangente. Assim, o repositório consolida um ponto de partida para quem deseja aprofundar o entendimento sobre riscos, confiabilidade e governança de sistemas de inteligência artificial.

O trabalho foi realizado na plataforma *Databricks*, consolidado inteiramente em um notebook (*mvp_ed.ipynb*) que foi exportado aqui para esse repositório.
